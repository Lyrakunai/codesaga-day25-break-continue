# Codesaga Day 25 - Break & Continue

## 📌 What I Learned
- break stops the loop completely
- continue skips current iteration
- execution flow is top to bottom
- order of conditions matters

---

## 🔹 break
Stops the loop immediately.

Example:
for i in range(5):
    if i == 3:
        break

---

## 🔹 continue
Skips current iteration and moves to next.

Example:
for i in range(5):
    if i == 2:
        continue

---

## 🔥 Key Difference

break → stops loop  
continue → skips iteration  

---

## ⚠️ Mistakes I Faced
- placing print before condition
- confusion about continue scope
- wrong execution order
- unnecessary continue

---

## 🧠 My Understanding
- continue only affects current iteration
- break ends entire loop
- conditions must come before output
- logic first, then print
